Welcome to Dropform documentation. Learn how to add working contact forms to any website in minutes, without any backend code.

New to Dropform? Start with our Quick Start Guide to get your first form working in under 5 minutes.

Quick Start

Get your contact form working in 3 simple steps:

1. Create Your HTML Form

Start with any HTML form. Keep your existing styling and structure.

Basic Contact Form
<form action="https://dropform.dev/submit/your-form-id" method="POST"> <input type="text" name="name" placeholder="Your Name" required> <input type="email" name="email" placeholder="your@email.com" required> <textarea name="message" placeholder="Your message..." required></textarea> <button type="submit">Send Message</button> </form>

2. Get Your Form ID

Sign up for Dropform and get your unique form ID. Replace your-form-id in the action URL above.

3. Start Receiving Emails

That's it! Form submissions will be delivered directly to your inbox instantly.

Pro Tip: Test your form by submitting it yourself. You should receive the email within seconds.

Basic Setup

Dropform works with any HTML form by simply pointing the action attribute to our endpoint.

Form Action URL

Your form action URL follows this pattern:

https://dropform.dev/submit/YOUR_FORM_ID

Replace YOUR_FORM_ID with the unique ID provided in your dashboard.

Required Attributes

  • action: Must point to your Dropform endpoint
  • method: Must be POST
  • name: Each input must have a name attribute

Form Fields

Dropform supports all standard HTML form fields:

Field Type HTML Description
Text Input <input type="text" name="name"> Single line text input
Email <input type="email" name="email"> Email with built-in validation
Textarea <textarea name="message"></textarea> Multi-line text input
Phone <input type="tel" name="phone"> Phone number input
Select <select name="subject">...</select> Dropdown selection
Checkbox <input type="checkbox" name="newsletter"> Single checkbox
Radio <input type="radio" name="type"> Radio button group

Custom Redirects

Redirect users to a custom thank you page after form submission:

<input type="hidden" name="_redirect" value="https://yoursite.com/thank-you">
Note: Redirect URLs must be on the same domain as your form for security reasons.

Spam Protection

Dropform includes built-in spam protection, but you can add a honeypot field for extra security:

Honeypot Anti-Spam
/* Hide this field with CSS */ <input type="text" name="_honeypot" style="display: none;" tabindex="-1">
